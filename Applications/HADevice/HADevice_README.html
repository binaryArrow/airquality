Home Automation Reference Applications

HA Applications included in the BitCloud SDK implement ZigBee PRO Home Automation profile. This sample code should be used as a template for custom applications that meet specific requirements and use cases. It also demonstrates common programming idioms and functionality that can and should be easily reused.

The application includes code implementing all device types necessary to form a fully functional HA network. To produce application image for a paticular device type uncomment appropriate define starting with APP_DEVICE_TYPE_ in configuration.h file and recompile the application. The following device types are available:

  • Combined Interface
  • Occupancy sensor
  • Dimmable light
  • Dimmable switch
A set of ready-to-use images are already built and provided with the SDK. The images are located in ./Evaluation Tools/HADevice directory.

Detailed application description and platform specific installation instructions are presented in BitCloud SDK Quick Start Guide.
